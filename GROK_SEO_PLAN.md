# COMPREHENSIVE SEO EXECUTION GUIDE FOR www.alumise.co.uk

**Goal**: Rank on Page 1 of Google for core terms (e.g. "aluminium bifold doors Edinburgh", "aluminium windows Edinburgh", "bifold doors Penicuik") and achieve 90–100% Ahrefs On-Page + Technical Score.

You are a Senior SEO Specialist. Execute this plan step-by-step with maximum precision.

### 1. Preparation & Access (Week 0)
- Gain full access to:
  - Google Search Console
  - Google Analytics 4
  - Google Business Profile
  - Website backend/hosting (Vercel/Netlify likely)
  - Ahrefs / Semrush
- Run these audits immediately:
  1. Full Ahrefs Site Audit
  2. Screaming Frog SEO Spider crawl
  3. PageSpeed Insights (mobile + desktop) on all key pages
  4. GSC Performance + Coverage + Core Web Vitals reports
- Document current baseline metrics in a shared Notion/Google Doc.

### 2. Technical SEO Fixes (Weeks 1–4 – Highest Priority)

**Critical Tasks**:
- Ensure proper **Server-Side Rendering** so meta tags are crawlable.
- Add unique, keyword-optimized:
  - Title Tags (55–60 characters)
  - Meta Descriptions (150–160 characters)
- Create and submit a complete **XML Sitemap**.
- Implement comprehensive **Schema Markup**:
  - Organization + LocalBusiness
  - Product schema on all product pages
  - Review + FAQPage schema
  - BreadcrumbList
- Optimize Core Web Vitals (LCP < 2.5s, CLS < 0.1, FID < 100ms):
  - Convert all images to WebP
  - Lazy loading + proper alt text
  - Minify JS/CSS
  - Use CDN
- Fix all crawl errors, duplicate content, thin pages, and canonical issues.
- Ensure fully responsive mobile-first design.

**Target**: Ahrefs Technical Score ≥ 95. All important pages indexed in GSC.

### 3. Keyword Research & Mapping (Week 1–2)

**Target Keyword Clusters**:
- Money Keywords: aluminium bifold doors Edinburgh, aluminium windows Edinburgh, bifold doors Penicuik, roof lanterns Edinburgh
- Product Keywords: slimline aluminium windows, heritage aluminium windows, curtain walling Scotland
- Informational: cost of bifold doors Edinburgh 2026, aluminium vs uPVC, planning permission roof lanterns

**Actions**:
- Export 200+ keywords from Ahrefs with volume, KD, and intent.
- Create a full keyword-to-page mapping (one primary keyword per page + supporting keywords).
- Build dedicated location pages for: Edinburgh, Penicuik, Midlothian, East Lothian, West Lothian, Livingston.

### 4. On-Page Optimization & Content Creation (Weeks 3–12)

**Homepage Optimization**:
- Strong hero section with primary keyword + clear CTA
- Trust signals, guarantees, and stats

**Product Pages** (Minimum 1,200–2,000 words each):
- Detailed technical specifications (U-values, security, finishes)
- Comparison tables (vs uPVC, timber, competitors)
- High-quality image galleries + before/after
- Installation process + case studies

**Content Hub (Blog)**:
- Publish 2–4 articles per month, examples:
  - “2026 Guide to Aluminium Bifold Doors in Edinburgh: Prices, Options & Installation”
  - “How Much Do Aluminium Windows Cost in Scotland in 2026?”
  - “Heritage Aluminium Windows – Classic Look with Modern Performance”

**On-Page Checklist (Apply to Every Page)**:
- H1 = Primary keyword
- Proper H2/H3 structure with secondary keywords
- Minimum 3–8 internal links per page
- Keyword-rich image alt text
- FAQ section at bottom
- Strong calls-to-action
